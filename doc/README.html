Thank you for purchasing this template !
You must do at least the first 2 steps to configure it, you will need only a few minutes to install the template.

1 - INSTALL

Upload all files and directories to your server.
You can remove the doc folder if you want.

2 - CUSTOMIZE

Update the animation settings in /resources/js/plugins.js
Line 1 : var _DENSITY = 20;
Line 2 : var _FPS = 30;
Line 3 : var _OPACITY = 0.50;

3 - ANALYTICS

Create a free account at google.com/analytics and generate your JavaScript code.

Now copy your JavaScript code in /resources/js/analytics.js

You can follow the page activity on google analytics dashboard.

4 - SITEMAP

Insert your website URL between location attribute in /sitemap.xml

5 - RATE THE TEMPLATE

Please take a few seconds to rate the template 5 stars on ThemeForest.

Thank you !
